## 时间的魔术师, 谷歌浏览器扩展程序


### 功能说明
* 记录访问的网站以及每个网站花费的时间
* 自动关闭黑名单中的网站  

###安装步骤
#### 1: git clone https://github.com/unasm/timeTracker.git

#### 2：更多工具 -> 打开chrome 扩展程序

#### 3: 将 目录下的crx文件拖进chrome 浏览器页面，松开之后 chrome 弹出 添加"时间的魔术师"  吗？ 点击 添加扩展
